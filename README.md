## Projects

**[Fraud Detection Pipeline & ML System](https://github.com/anirudhcancode/fraud-detection-pipeline)**
End-to-end fraud detection over 284,807 real credit card transactions. PySpark ETL with behavioral feature engineering, Random Forest and XGBoost classifiers, deployed via FastAPI on Docker. ROC-AUC of 0.98.

**[LLM Financial Report Analyzer](https://github.com/anirudhcancode/llm-financial-analyzer)**
NLP pipeline that summarizes earnings call transcripts and classifies financial sentiment. BART for abstractive summarization, FinBERT for domain-specific sentiment, YAKE for keyword extraction. Correctly identified Apple as the only negative report in the test set at 94% confidence.

**[A/B Testing & Experimentation Platform](https://github.com/anirudhcancode/ab-testing-pipeline)**
Full experimentation pipeline simulating a 10,000-user e-commerce checkout A/B test over 14 days. Z-test for significance, confidence intervals, lift metrics, and a REST API with on-demand simulation. Detected a 26% conversion lift at p = 0.000007.

**AI Companion** *(private — [project overview](https://anirudhcancode.github.io/portfolio/ai-companion-demo.html))*
Cross-platform AI assistant (iOS, Android, web) built around persistent memory, emotional awareness, and cross-domain synthesis. Two-layer architecture: tool-orchestrated reasoning that decides what context a moment calls for, and a mind layer that maintains a continuously updated understanding of the person. Trust-gated disclosure model governs what the system voices versus what it holds privately. Live in production.
